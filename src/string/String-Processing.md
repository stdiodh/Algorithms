# String Processing (문자열 처리)

## 개요
문자열 처리 알고리즘은 문자열을 다루는 다양한 기법들을 포함합니다.

## 핵심 개념
- 문자열 매칭, 파싱
- 정규식 활용
- 문자열 변환

## 복잡도
- **시간 복잡도**: O(n) ~ O(n²)
- **공간 복잡도**: O(n)

## 주요 활용 사례
- 문자열 검색
- 파싱
- 문자열 변환
