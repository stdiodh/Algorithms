# DFS (깊이 우선 탐색)

## 개요
DFS는 그래프를 깊이 우선으로 탐색하는 알고리즘입니다. 스택 또는 재귀를 사용합니다.

## 핵심 개념
- 스택(Stack) 또는 재귀 사용
- 백트래킹 문제에 적합
- 경로 탐색, 사이클 탐지에 활용

## 복잡도
- **시간 복잡도**: O(V + E)
- **공간 복잡도**: O(V)

## 주요 활용 사례
- 경로 탐색
- 사이클 탐지
- 트리 순회

- [1012번](https://www.acmicpc.net/problem/1012) - 1012.kt
- [11725번](https://www.acmicpc.net/problem/11725) - 11725.kt
