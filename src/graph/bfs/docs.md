# BFS (너비 우선 탐색)

## 개요
BFS는 그래프를 레벨별로 탐색하는 알고리즘입니다. 큐를 사용하여 같은 레벨의 노드들을 먼저 방문합니다.

## 핵심 개념
- 큐(Queue) 자료구조 사용
- 최단 경로 문제에 적합
- 레벨별 탐색으로 거리 계산 가능

## 복잡도
- **시간 복잡도**: O(V + E)
- **공간 복잡도**: O(V)

## 주요 활용 사례
- 미로 탐색
- 최단 경로 찾기
- 연결 요소 찾기
