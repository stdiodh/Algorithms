# Greedy Algorithm (그리디 알고리즘)

## 개요
그리디 알고리즘은 각 단계에서 최선의 선택을 하는 방식으로 문제를 해결합니다.

## 핵심 개념
- 매 순간 최선의 선택
- 전역 최적해를 보장하지 않을 수 있음
- 활동 선택, 최소 신장 트리 등에 활용

## 복잡도
- **시간 복잡도**: O(n log n) (정렬 포함)
- **공간 복잡도**: O(1) ~ O(n)

## 주요 활용 사례
- 활동 선택 문제
- 최소 비용 문제
- 간격 스케줄링
