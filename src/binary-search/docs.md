# Binary Search (이진 탐색)

## 개요
이진 탐색은 정렬된 배열에서 특정 값을 효율적으로 찾는 알고리즘입니다.

## 핵심 개념
- 정렬된 데이터에서만 사용 가능
- 매 단계마다 탐색 범위를 절반으로 줄임
- 매우 빠른 탐색 속도

## 복잡도
- **시간 복잡도**: O(log n)
- **공간 복잡도**: O(1)

## 주요 활용 사례
- 정렬된 배열에서 값 찾기
- 파라메트릭 서치
- 최적값 찾기
