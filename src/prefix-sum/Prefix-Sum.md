# Prefix Sum (누적합)

## 개요
누적합은 배열의 각 위치까지의 합을 미리 계산하여 구간 합을 빠르게 구하는 기법입니다.

## 핵심 개념
- 구간 합을 O(1)에 계산 가능
- 1차원, 2차원 배열 모두 적용 가능
- 구간 쿼리 문제에 유용

## 복잡도
- **시간 복잡도**: O(n) (전처리), O(1) (쿼리)
- **공간 복잡도**: O(n)

## 주요 활용 사례
- 구간 합 계산
- 2D 배열 부분 합
- 구간 쿼리 최적화
