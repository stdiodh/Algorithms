# Hash Map (해시맵)

## 개요
해시맵은 키-값 쌍을 저장하는 자료구조로, 평균적으로 O(1)의 접근 시간을 제공합니다.

## 핵심 개념
- 키를 해시 함수로 변환하여 저장
- 평균 O(1) 접근 시간
- 중복 체크, 카운팅에 유용

## 복잡도
- **시간 복잡도**: O(1) 평균, O(n) 최악
- **공간 복잡도**: O(n)

## 주요 활용 사례
- 빠른 조회
- 중복 체크
- 카운팅
