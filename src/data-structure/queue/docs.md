# Queue (큐)

## 개요
큐는 FIFO(First In First Out) 원칙을 따르는 선형 자료구조입니다.

## 핵심 개념
- 선입선출 구조
- enqueue, dequeue 연산이 O(1)
- BFS, 스케줄링에 활용

## 복잡도
- **시간 복잡도**: O(1) (enqueue/dequeue)
- **공간 복잡도**: O(n)

## 주요 활용 사례
- BFS 구현
- 작업 스케줄링
- 버퍼 관리
