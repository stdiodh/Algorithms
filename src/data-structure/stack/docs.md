# Stack (스택)

## 개요
스택은 LIFO(Last In First Out) 원칙을 따르는 선형 자료구조입니다.

## 핵심 개념
- 후입선출 구조
- push, pop 연산이 O(1)
- 괄호 매칭, 후위 표기법 등에 활용

## 복잡도
- **시간 복잡도**: O(1) (push/pop)
- **공간 복잡도**: O(n)

## 주요 활용 사례
- 괄호 매칭
- 후위 표기법 계산
- 역순 처리
