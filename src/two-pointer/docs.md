# Two Pointer (투 포인터)

## 개요
투 포인터는 배열이나 문자열에서 두 개의 포인터를 사용하여 효율적으로 문제를 해결하는 기법입니다.

## 핵심 개념
- 두 개의 포인터를 조작하여 탐색
- O(n²)를 O(n)으로 최적화 가능
- 정렬된 배열에서 특히 유용

## 복잡도
- **시간 복잡도**: O(n)
- **공간 복잡도**: O(1)

## 주요 활용 사례
- 합이 특정 값인 쌍 찾기
- 부분 배열 문제
- 정렬된 배열 처리
