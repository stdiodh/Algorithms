# Parsing (파싱)

## 개요
파싱은 입력 데이터를 분석하고 구조화하는 과정입니다.

## 핵심 개념
- 입력 형식 분석
- 정규식 활용
- 데이터 추출 및 변환

## 복잡도
- **시간 복잡도**: O(n)
- **공간 복잡도**: O(n)

## 주요 활용 사례
- 로그 파싱
- 데이터 추출
- 형식 변환
