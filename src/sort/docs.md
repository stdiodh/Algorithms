# Sorting (정렬)

## 개요
정렬은 데이터를 특정 순서대로 나열하는 알고리즘입니다.

## 핵심 개념
- 다양한 정렬 알고리즘 존재
- 시간 복잡도: O(n log n) 최적
- 정렬 후 이진 탐색 등 활용

## 복잡도
- **시간 복잡도**: O(n log n) (최적)
- **공간 복잡도**: O(1) ~ O(n)

## 주요 활용 사례
- 데이터 정렬
- 정렬 후 처리
- 중복 제거
