# Sliding Window (슬라이딩 윈도우)

## 개요
슬라이딩 윈도우는 고정된 크기의 윈도우를 이동시키며 문제를 해결하는 기법입니다.

## 핵심 개념
- 고정 또는 가변 크기 윈도우
- 중복 계산을 피하여 효율성 향상
- 연속된 부분 배열/문자열 문제에 적합

## 복잡도
- **시간 복잡도**: O(n)
- **공간 복잡도**: O(k) (윈도우 크기)

## 주요 활용 사례
- 최대/최소 부분 배열
- 고유 원소 개수
- 부분 문자열 문제
