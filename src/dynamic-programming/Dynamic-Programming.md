# Dynamic Programming (동적 프로그래밍)

## 개요
DP는 큰 문제를 작은 하위 문제로 나누어 해결하고, 결과를 저장하여 재사용하는 알고리즘 기법입니다.

## 핵심 개념
- 메모이제이션 또는 타뷸레이션 사용
- 중복 계산을 피하여 효율성 향상
- 최적 부분 구조와 중복 부분 문제 조건 필요

## 복잡도
- **시간 복잡도**: O(n²) ~ O(n³) (문제에 따라 다름)
- **공간 복잡도**: O(n) ~ O(n²)

## 주요 활용 사례
- 최적화 문제
- 경로의 개수 계산
- 최대/최소 값 찾기
